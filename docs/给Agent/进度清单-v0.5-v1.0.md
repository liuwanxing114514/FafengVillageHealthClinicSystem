## 每版标准流程

```text
1. git checkout v0.X-xxx && git merge develop
2. 实现本版（严守 ROADMAP，不做下一版）
3. 自测：mvn test + 前端 build + 手工验收
4. git push origin v0.X-xxx
5. git checkout develop && git merge v0.X-xxx && git push origin develop
6. 更新本清单对应行
```
