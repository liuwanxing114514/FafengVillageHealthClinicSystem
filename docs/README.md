# 文档总览

发凤村卫生室诊所辅助系统的全部说明文档。**请先判断您是谁，再进对应目录。**

```text
docs/
├── README.md                 ← 您在这里
├── 给人看/                   ← 诊所使用者、您本人日常查阅
│   ├── README.md
│   ├── 使用者功能确认.md     系统做什么、怎么用
│   └── 后续版本操作说明.md   一期完成后，第二/三/…期怎么上
├── 给Agent/                  ← Cursor 开发时用
│   ├── README.md
│   ├── 开发交接.md           第一期总原则、不可破坏规则
│   ├── 版本任务指南.md       每版本 Prompt + 版本速查表（分支/进度/开对话）
│   ├── 功能点与实现记录.md   各版本进度与当前工作点
│   └── AI架构.md             AI/脱敏/草稿/Agent 技术约定
└── 共用/                     ← 人与 Agent 都对齐这里
    ├── README.md
    ├── ROADMAP.md            ★ 版本号唯一权威
    ├── 大纲.md               业务与技术总纲
    └── DEPLOYMENT.md         部署、备份、升级命令
```

## 我是谁，该读什么？

### 诊所医生 / 管理员（不看代码）

1. [`给人看/使用者功能确认.md`](给人看/使用者功能确认.md) — 功能与日常流程  
2. [`给人看/后续版本操作说明.md`](给人看/后续版本操作说明.md) — 新功能上线后您要做什么  
3. [`给人看/使用者功能确认.md`](给人看/使用者功能确认.md) 第十八节 — 后续功能预告（白话）

**不必读**：`给Agent/` 下任何文件。

---

### 项目负责人（您自己：定需求 + 指挥 Cursor + 诊所升级）

| 阶段 | 读什么 |
| --- | --- |
| 了解全貌 | [`共用/大纲.md`](共用/大纲.md) |
| 查版本范围 | [`共用/ROADMAP.md`](共用/ROADMAP.md) |

| 诊所电脑部署/升级 | [`共用/DEPLOYMENT.md`](共用/DEPLOYMENT.md) + [`给人看/后续版本操作说明.md`](给人看/后续版本操作说明.md) |
| 开 AI 功能前 | [`给Agent/AI架构.md`](给Agent/AI架构.md) 脱敏规则 |
